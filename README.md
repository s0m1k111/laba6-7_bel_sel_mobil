# Tip Time — Калькулятор чаевых

Мобильное приложение для Android, которое помогает быстро рассчитать сумму чаевых в ресторане или кафе. Пользователь вводит сумму счёта и желаемый процент чаевых, приложение автоматически вычисляет итоговую сумму и при необходимости округляет результат.

## Цель работы

Изучение концепции состояния (state) в Android-приложениях и практическое применение State в Jetpack Compose. В ходе лабораторной работы были освоены:
- работа с пользовательским вводом через TextField;
- подъём состояния (State Hoisting);
- создание переиспользуемых компонентов;
- автоматизированное тестирование Android-приложений.

## Функциональность

Приложение реализует следующий функционал:

- **Ввод суммы счета** — текстовое поле с числовой клавиатурой и иконкой денег
- **Ввод процента чаевых** — текстовое поле с числовой клавиатурой и иконкой процента
- **Автоматический расчет** — мгновенное вычисление суммы чаевых при изменении любого поля
- **Округление чаевых** — переключатель (Switch) для округления результата в большую сторону
- **Локализация** — поддержка форматирования валюты в зависимости от региона устройства
- **Адаптивный интерфейс** — поддержка портретной и альбомной ориентации с прокруткой
- **Оптимизация ввода** — клавиша Next для перехода между полями и Done для закрытия клавиатуры

## Технологии

Проект разработан с использованием современного стека технологий:

- **Язык программирования:** Kotlin
- **UI Toolkit:** Jetpack Compose
- **Дизайн-система:** Material Design 3 
- **Среда разработки:** Android Studio
- **Сборка проекта:** Gradle

    - Compose UI Testing для инструментальных тестов
